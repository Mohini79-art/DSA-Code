## 🚀 Projects Included  

### 📝 Java Quiz Project  
A console-based quiz game for practicing Java basics.  

✅ **Features:**  
- 5 MCQ Questions  
- Correct/Incorrect feedback  
- Final Score Display  

▶ **Run:**  
```bash
cd Projects
javac JavaQuiz.java
java JavaQuiz
```

---

### 💳 ATM Simulation Project  
A basic ATM simulation program for learning OOP concepts.  

✅ **Features:**  
- Balance Check  
- Withdraw/Deposit  
- PIN Verification  

▶ **Run:**  
```bash
cd Projects
javac ATM.java
java ATM
```
